# Security Policy

## Supported Versions

Currently, we only provide security updates for the latest version of React Toast MSG.

| Version | Supported          |
| ------- | ------------------ |
| >= 2.x  | :white_check_mark: |
| < 2.x   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability, please do NOT open a public issue. Instead, report it privately.

Please email the lead maintainer at [contact@sudhucodes.com](mailto:contact@sudhucodes.com) with detailed information about the vulnerability. We aim to respond within 48 hours.

Once a fix is prepared, we will release a security advisory and a patched version as soon as possible.
