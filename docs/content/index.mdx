Welcome to the **react-toast-msg** documentation!

react-toast-msg is designed to be the simplest and most flexible way to add toast notifications to your React applications. It provides a clean API, sensible defaults and full customization options using Tailwind CSS.

## Key Features

- **Fast Setup**: Get started in less than a minute.
- **Tailwind CSS**: Styled with Tailwind for easy customization and modern look.
- **Lightweight**: Zero unnecessary bloat.
- **TypeScript**: Built with TypeScript for excellent developer experience.
- **Customizable**: Control durations, types and more.
