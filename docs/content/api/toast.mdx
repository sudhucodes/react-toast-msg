The `toast` object provides several methods to trigger notifications with different styles.

## toast(message, type, duration)

The main function for showing a toast.

<table>
    <thead>
        <tr>
            <th>Parameter</th>
            <th>Type</th>
            <th>Default</th>
            <th>Description</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>message</td>
            <td>string</td>
            <td>-</td>
            <td>The message to display.</td>
        </tr>
        <tr>
            <td>type</td>
            <td>ToastType</td>
            <td>'default'</td>
            <td>The style of the toast.</td>
        </tr>
        <tr>
            <td>duration</td>
            <td>number</td>
            <td>3000</td>
            <td>Time in ms before auto-close.</td>
        </tr>
    </tbody>
</table>

## Methods

### toast.success(message, duration?)

Shorthand for `toast(message, duration)`.

### toast.error(message, duration?)

Shorthand for `toast.error(message, duration)`.

### toast.warning(message, duration?)

Shorthand for `toast.warning(message, duration)`.

### toast.default(message, duration?)

Shorthand for `toast(message, duration)`.

## Example

```tsx copyable
import { toast } from "react-toast-msg";

toast.success("Profile updated", { duration: 2000 });
```
