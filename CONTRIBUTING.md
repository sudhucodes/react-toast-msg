# Contributing to React Toast MSG

First off, thank you for considering contributing to React Toast MSG! It's people like you who make it such a great tool.

## Code of Conduct

Help us keep React Toast MSG open and inclusive. Please read and follow our [Code of Conduct](CODE_OF_CONDUCT.md).

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18 or higher)
- [pnpm](https://pnpm.io/) (v9 or higher)

### Installation

1. Fork the repository.
2. Clone your fork:
    ```bash
    git clone https://github.com/sudhucodes/react-toast-msg.git
    ```
3. Install dependencies:
    ```bash
    pnpm install
    ```

## Development Workflow

### Running tests

We use Vitest for unit testing. Please ensure all tests pass before submitting a pull request:

```bash
pnpm test
```

### Building the project

To verify the build and type generation:

```bash
pnpm build
```

### Style

We use Prettier for code formatting. You can run it manually:

```bash
pnpm prettier
```

## Submitting changes

1. Create a new branch for your feature or bugfix.
2. Make your changes and add tests if applicable.
3. Ensure the project builds and all tests pass.
4. Commit your changes with descriptive messages.
5. Push to your fork and submit a pull request to the `main` branch.

## Questions?

If you have any questions, feel free to open an issue or reach out to the maintainers.

Happy coding!
