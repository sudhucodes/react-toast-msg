## Quick Start

1. Add `ToastContainer` to your application:

```tsx copy
import { ToastContainer } from 'react-toast-msg';

function App() {
    return (
        <>
            {children}
            <ToastContainer />
        </>
    );
}
```

2. Use the `toast` function to show notifications:

```tsx copy
import { toast } from 'react-toast-msg';

toast.success('Profile updated');
```
